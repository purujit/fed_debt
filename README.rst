========
fed_debt
========


    Forecast Federal Debt and Interest Expense.


Use a data dump from the Treasury department that lists all existing federal debt
to forecast future debt level under different configurations of spending and revenue growth
and interest rate regime.


.. _pyscaffold-notes:

Making Changes & Contributing
=============================

This project uses `pre-commit`_, please make sure to install it before making any
changes::

    pip install pre-commit
    cd fed_debt
    pre-commit install

It is a good idea to update the hooks to the latest version::

    pre-commit autoupdate

Don't forget to tell your contributors to also install and use pre-commit.
